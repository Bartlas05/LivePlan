# 📅 Live Plan

Aplikacja z interaktywnym planem zajęć.

---

## Funkcje

* **Live widget** – inforamcje o aktualnych zajęciach
* **Plan lekcji** – przejrzysty harmonogram tygodniowy
* **Harmonogram zjazdów** – rozkład zjazdów w danym semestrze
* **Dodawanie wydarzeń** – dodaj własne notatki do konkretnych zajęć
---

## Technologie

* HTML
* CSS
* JavaScript
* JSON

---

## 🚀 Instalacja

```bash
git https://github.com/Bartlas05/LivePlan
npm install
npm run dev
```

## Struktura projektu

```
src/
 ├── Icons/              # zasoby graficzne
 ├── netlify/functions/  # funkcje serverless
 ├── changelog.json      # plik z historią zmian
 ├── index.html          # główny plik HTML
 ├── manifest.json       # konfiguracja aplikacji (PWA)
 ├── plan.json           # rozkłąd zajęć 
 ├── raport.html         # fomularz do zgłaszania błędów
 ├── script.js           # główny plik logiki JavaScript
 ├── settings.html       # ustawienia
 └── ui.css              # główny arkusz stylów
```
